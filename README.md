Silex with Twitter Bootstrap application template
========================

Silex application template 
+ Twig 
+ FluentPDO 
+ JQuery 
+ JQuery UI 
+ Twitter Bootstrap 
+ Font Awesome

Setup
---------------

### 1) composer

    composer update

### 2) config.php

    cp src/app/config.php.dist src/app/config.php

### 3) Database

Require database or PDO throw exception. See src/config.php.
